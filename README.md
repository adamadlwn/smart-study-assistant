# Smart Study Assistant (SSA)

Project UAS Praktikum Pemrograman Web Lanjut  
Tim: Code Thirteen Dev (Kelompok 13)

## Deskripsi
Aplikasi pembelajaran berbasis web menggunakan Django dan AI untuk membantu mahasiswa belajar lebih efektif.

## Fitur Utama
- AI Summary
- AI Q&A
- Quiz Generator

## Teknologi
- Python Django
- HTML/CSS
- AI API
- SQLite
